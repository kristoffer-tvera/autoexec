How to use this tool?

  1. Adjust the settings.
  2. Click "Update" below. You'll see text fill into the text field below.
  3. Manually add any settings if you need.
  4. Click "Download" below. Your browser will prompt to open/save autoexec.cfg.
  5. Place the file in your CS:GO cfg folder - \Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg
